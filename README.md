# 🩺 PulseTalk – AI-Powered Healthcare Communication Platform

PulseTalk is a modern healthcare communication platform designed to bridge the gap between patients and healthcare professionals through intelligent, real-time interactions.

The platform combines AI-powered assistance, appointment management, multilingual communication, patient monitoring, and clinician analytics into a unified healthcare ecosystem.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/PulseTalk.git
cd PulseTalk
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env.local` file:

```env
VITE_GEMINI_API_KEY=YOUR_API_KEY
```

### Run Development Server

```bash
npm run dev
```
